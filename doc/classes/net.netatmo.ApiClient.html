Constructor: Initialize a NetAtmo OAuth2.0 Client.

__construct(\array $config = array())

Arguments

$config

\array

An associative array as below:

  • code: (optional) The authorization code.
  • username: (optional) The username.
  • password: (optional) The password.
  • client_id: (optional) The application ID.
  • client_secret: (optional) The application secret.
  • refresh_token: (optional) A stored refresh_token to use
  • access_token: (optional) A stored access_token to use
  • object_cb : (optionale) An object for which func_cb method will be applied if object_cb exists
  • func_cb : (optional) A method called back to store tokens in its context (session for instance)

Make an API call.

api(string $path, string $method = 'GET', \array $params = array(), \bool $secure = false) : \array

Support both OAuth2.0 or normal GET/POST API call, with relative or absolute URI.

If no valid OAuth2.0 access token found in session object, this function will automatically switch as normal remote API call without "access_token" parameter.

Assume server reply in JSON object and always decode during return. If you hope to issue a raw query, please use makeRequest().
